## 제작 동기

	- 강아지 발바닥 체크, 퍼즐 완성, 연말 전시회까지 이어지는 게임 루프 구현
	- iOS / Android 동시 출시
	- 광고 수익화

---

## 주요 작업

- 습관 CRUD + 발바닥 체크·완료 도장 애니메이션
- 달력 히트맵
- 퍼즐 조각 획득 & easy 난이도 그림 완성
- 강아지 케어 (밥/산책/목욕) + 상태 변화 비주얼
- 강아지 아픔 상태 + 치료 기능
- 신규 강아지 보호 기간 — 입양 후 7일간은 아프지 않게 해 초반 진입 문턱 완화
- 상점·스티커·슬롯으로 이어지는 포인트 경제
- 슬롯 구매 게이트 — 무료 9칸을 다 채우면 광고로 슬롯 확장
- 리워드 광고 수익화 — 일일 3회 캡, AdMob 실 광고 적용
- 전면 UI 리디자인(Claude Design) — 파스텔 색 개편, 회고(연속 달성일) 뷰
- 마을 트랙 지도 — 집에서 표지판으로 이어지는 산책 동선
- 완료 진동(햅틱) on/off 설정
- 로컬 저장 (Drift)
- JSON / CSV Export · Import

- 남은 일은 스토어 에셋·개인정보처리방침·실기기 검증 정도로, 출시 직전 단계.
