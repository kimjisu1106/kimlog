## 오늘 한 일

- 합성음을 실제 피아노 샘플(Salamander Grand Piano)로 교체 — 배음을 쌓아 만든 합성음 대신 실제로 녹음된 피아노 샘플을 쓰게 함
	- 이유: 합성음은 아무리 다듬어도 진짜 피아노 질감이 안 났다. Salamander는 CC-BY 무료 음원이라 출처만 표기하면 상업적으로도 쓸 수 있음
- 서스테인 페달 반영 — 페달을 밟은 채 건반을 떼도 페달을 뗄 때까지 소리가 이어지게. 페달을 짧게 뗐다 곧바로 다시 밟는 구간에서 소리가 끊기던 것도 이어지게
- 소리 고르기 — 같은 세기로 쳤는데 음마다 크기가 들쭉날쭉하던 것, 같은 음을 연달아 칠 때 소리가 겹쳐 두 배로 커지던 것, 화음이냐 단음이냐에 따라 전체 볼륨이 출렁이던 것을 바로잡음
- 진짜 백그라운드 재생 — 다른 탭으로 옮겨도 소리가 안 끊기게 오디오를 미리 예약해 두는 방식으로. 돌아왔을 때 밀린 소리가 한꺼번에 터지던 것도 막음
- 프로젝트 파일에 오디오까지 담기 — `.kbloom` 파일 하나에 오디오 원본을 넣어, 그 파일만 있으면 어디서든 그대로 열리게 함(네이티브 앱 대비)
- 무료 버전 워터마크 — 내보낸 영상 위쪽에 KeyBloom 표시를 항상 박음. 미리보기 화면에도 늘 띄워서 화면 녹화로 우회하지 못하게
- 오디오 소스 선택 — 프로젝트에 오디오가 들어 있어도 파일 / 샘플 피아노 / 무음 중에 골라 미리듣기·내보내기 할 수 있게
- 탭 정리 — 파일(앱 공통 저장·설정) / 프로젝트(이 곡의 소스·출력) / 비주얼로 기준을 나누고, 내보내기를 프로젝트 탭으로 옮김
- 출시 채비 — 정보 섹션에 제작자·음원·오픈소스 크레딧, 공유 카드(OG)와 태그라인 "연주가 피어나는 순간", 앱 아이콘, 영어 지원(브라우저 언어 자동 감지 + 토글), 인앱 도움말(사용법 + "프로젝트는 서버에 저장되지 않으니 파일로 직접 보관" 고지)
- Cloudflare Pages 배포 + 카카오 광고 부착 — 무료 버전 출시

## 막힌 부분

- 예전에 저장한 `.kbloom` 파일이 파일 선택창에서 아예 안 잡혔다
	- 원인: 브라우저가 커스텀 확장자를 특정 형식(JSON)으로 묶어 회색 처리함
	- 해결: 파일 형식 지정을 범용(옥텟 스트림)으로 바꿔 선택 가능하게
- 같은 음을 연달아 치면 소리가 겹쳐 두 배로 크게 났다
	- 해결: 이전 소리를 짧게 페이드로 끄고 새 소리를 내게
- 특정 화음(0:26)에서 서스테인이 끊기는 느낌이 남았다
	- 페달 재밟기 구간을 이어 붙이는 처리까지 넣었는데도 청감상 잔존, 원인 미확정 → 백로그로 남김
