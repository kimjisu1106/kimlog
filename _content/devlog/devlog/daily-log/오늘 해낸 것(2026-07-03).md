## 배운 점

- (소리꽃 KeyBloom TIL 1로 정리) `canvas.captureStream` + `MediaRecorder`로 캔버스를 브라우저 안에서 영상으로 저장 / `<audio>`의 `captureStream`으로 오디오 트랙을 합쳐 소리까지 녹음 / `clearRect`로 그리면 알파 WebM(투명 배경) 저장
- (소리꽃 KeyBloom TIL 2로 정리) `fillText`는 매번 폰트 파싱·글리프 래스터화라 비싸다 → 미리 색 틴트한 스프라이트를 `drawImage`로 / 색은 24단계 양자화해 `colorT` 인덱스 / 광택은 `source-atop`·`lighter` 합성 / 제자리 압축으로 GC 절감
- (소리꽃 KeyBloom TIL 3으로 정리) 여러 모션을 물리값(속도·중력·감쇠·흔들림·수명)으로 환원해 하나의 제네릭 update로 / 토네이도=흔들림 진폭 성장(swayGrow) / 불꽃=파티클이 파티클을 낳는 2단계, 폭발은 루프 뒤에 방출
- (소리꽃 KeyBloom TIL 4로 정리) 위치를 픽셀 대신 0~1 비율로 저장하고 픽셀 변환은 렌더에서만(해상도·비율 독립) / 16:9 레터박스 뷰 / 캔버스 DPR로 선명하게
- (소리꽃 KeyBloom TIL 5로 정리) Web Audio 오실레이터로 샘플 없이(오프라인) 합성음 / MIDI→주파수 `440*2^((n-69)/12)` / 게인 엔벨로프 목표값은 0 대신 0.0001(지수 램프·클릭 방지)
- (여둘까 Office Layout TIL 4로 정리) 캡처 단계 리스너 + `stopPropagation`으로 남의 이벤트 핸들러를 안 건드리고 특정 제스처만 가로채기 / 브라우저의 pointer·mouse 이중 이벤트 계열 / 짝 이벤트 중복·잔여 처리
