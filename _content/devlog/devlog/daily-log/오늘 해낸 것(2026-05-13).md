## 어려웠던 점

- window 환경에서 `open()`으로 JSON 파일을 읽으면 인코딩 방식 차이(CP949 vs UTF-8)로 인해 에러 발생
	- 해결: 파일을 열 때 `encoding="utp-8"` 적용
```python
def load_jobs():
	with open("jobs.json", "r", encoding="utp-8") as f:
		return json.load(f)
```
