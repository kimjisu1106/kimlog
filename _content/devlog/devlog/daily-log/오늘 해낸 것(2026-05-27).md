## 오늘 한 일

- 회사 툴
	- SharePoint 공유 링크 검토 기능
		- 이유: 일부 폴더 URL이 `AllItems.aspx` 포함 URL 형식이었음. `AllItems.aspx` URL은 절대경로가 아닌 뷰 기반 URL이라 폴더 경로 변경 시 접근 불가.
		- 방안: `AllItems.aspx` URL 포함 시 API 호출 없이 즉시 오류 판정. 권한 이슈로 판정된 것을 직접 URL 변경 중.
	- 월간 예산/실제 비교표 구조 개선
		- 이유: 파트장님이 Budget/Actual 2행 구조 요청.
		- 문제점: 엑셀처럼 `rowspan`을 사용해 셀 병합을 하자 오히려 예외사항 처리가 어렵고 복잡해짐.
		- 방안: Budget/Actual을 아코디언 그룹 구조로 만들고 Excel export 동기화 함.
	- 🌟오류 로그에 내가 만들지 않은 오류가 걸렸다. 팀원이 사용하다가 생긴 오류를 수정하는데 오류 로그 만든건 정말 잘한 것 같다는 생각을 했다(오류 내용은 변수명 오타).
