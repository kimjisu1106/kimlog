변수에 어떤 종류의 데이터를 저장할지 미리 정해주는 것. C++는 변수를 만들 때 반드시 자료형을 선언해야 한다.

---

### 정수형 — 소수점 없는 숫자

| 자료형            | 크기   | 범위                                                     |
| -------------- | ---- | ------------------------------------------------------ |
| `int`          | 4바이트 | -2,147,483,648 ~ 2,147,483,647                         |
| `short`        | 2바이트 | -32,768 ~ 32,767                                       |
| `long long`    | 8바이트 | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 |
| `unsigned int` | 4바이트 | 0 ~ 4,294,967,295 (음수 없음)                              |

- `long`
	- 플랫폼에 따라서 크기가 달라지기 때문에 `long`보다는 `int32_t`(4바이트), `int64_t`(8바이트), `long long`(8바이트)사용. 
	- Windows 64bit에서는 4바이트(`int`랑 같은 크기), Linux/macOS 64bit에서는 8바이트(`long long`과 같은 크기)

```cpp
int hp = 100;
int damage = -30;
```

---

### 실수형 — 소수점 있는 숫자

|자료형|크기|정밀도|
|---|---|---|
|`float`|4바이트|소수점 7자리|
|`double`|8바이트|소수점 15자리|

```cpp
float speed = 3.14f;   // f를 붙여야 float. 안붙이면 double로 인식.
double ratio = 3.141592653589;
```

---

### 문자형

| 자료형      | 크기   | 설명                |
| -------- | ---- | ----------------- |
| `char`   | 1바이트 | 문자 1개 (알파벳, 숫자 등) |
| `string` | 가변   | 문자 여러 개 (문자열)     |

```cpp
char grade = 'A';       // 작은따옴표
string name = "김로그"; // 큰따옴표
```

---

### 논리형

| 자료형    | 크기   | 값                |
| ------ | ---- | ---------------- |
| `bool` | 1바이트 | `true` / `false` |

```cpp
bool isAlive = true;
bool isDead = false;
```

---

### 왜 자료형을 나누는가?

딱 필요한 크기만 할당하여 메모리를 효율적으로 쓰기 위함.
