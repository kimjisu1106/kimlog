### Python 변수 선언

```python
top_average = 0
top_title = "" # None으로 선언해도 됨
highly_rated_count = 0
sum_rating = 0
```

---
### Python `for` 문법

`for item in list:` 로 요소를 직접 꺼냄, 인덱스 불필요

```python
for movie_id in movie_ids:
    response = requests.get(f"{movie_urls}movies/{movie_id}")
    data = response.json()
    
    print(f"Title: {data['title']}\n Overview: {data['overview']}\n Vote average: {data['vote_average']}\n =========================")
    
    if top_average < data["vote_average"]:
       top_average = data["vote_average"]
       top_title = data["title"]
    if data["vote_average"] > 7:
       highly_rated_count += 1
    sum_rating += data["vote_average"]
```

---

### `requests` 모듈

HTTP 요청을 보내는 모듈. `requests.get(url)` 로 API 호출
```python
response = requests.get(f"{movie_urls}movies/{movie_id}")
```

---

### API의 응답 형태

| 응답 형태                                  | Python 타입 | 접근 방법           | 예시      |
| -------------------------------------- | --------- | --------------- | ------- |
| `{"title": "..."}`                     | `dict`    | `data["title"]` | 오늘 한 것  |
| `[{"title": "..."}, {"title": "..."}]` | `list`    | `data[0]`       | 영화 목록   |
| `"hello"`                              | `str`     | `data`          | 드물지만 있음 |

---

### 오늘 사용한 Python 내장 함수

- `print()`: `print(f"Top title: {top_title}")`
- `type()`: `print(type(data))`
