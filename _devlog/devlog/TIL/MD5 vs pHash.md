### MD5 (완전 동일 탐지)

MD5는 **암호학적 해시 함수**로, 파일의 모든 바이트를 입력으로 받아 128비트(32자리 16진수) 고정 길이 문자열을 출력.

- 파일 내용이 **1비트라도 다르면** 완전히 다른 해시값이 나옴 (눈사태 효과)
- 반대로 해시가 같으면 파일 내용이 동일하다고 볼 수 있음
- 8KB 청크씩 읽어가며 해시를 누적 계산 → 메모리 효율적

```
파일 A (1GB) → MD5 → "d41d8cd98f00b204e9800998ecf8427e"
파일 B (1GB) → MD5 → "d41d8cd98f00b204e9800998ecf8427e"  ← 동일!
```

**용도:** 이름이 달라도, 위치가 달라도 내용이 완전히 같은 파일 탐지

### Perceptual Hash (유사 이미지 탐지)

pHash는 **지각적 해시**로, 이미지를 사람 눈에 "비슷해 보이는지" 기준으로 비교.

1. 이미지를 작게 축소(32×32 등)
2. DCT(이산 코사인 변환) 적용 → 고주파 노이즈 제거, 저주파 구조만 추출
3. DCT 계수의 평균값보다 크면 1, 작으면 0으로 64비트 해시 생성

```
원본 이미지 → pHash → 1011001101...  (64비트)
약간 압축된 동일 이미지 → pHash → 1011001101...  ← 거의 동일!
```

두 해시 간 **해밍 거리** (다른 비트 수)로 유사도 측정:

- 거리 0 = 완전 동일
- 거리 10 이하 = 유사 (앱 기본값)
- 거리 커질수록 = 더 다름

**용도:** 리사이즈, 압축률 변경, 경미한 편집이 된 같은 사진 탐지 (MD5로는 다른 파일로 인식됨)
