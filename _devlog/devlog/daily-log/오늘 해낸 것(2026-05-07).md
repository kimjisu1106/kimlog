## 오늘 한 일

- 회사 툴
	- draft_form vendor manual 기능 추가
		- 문제점: draft의 vendor가 default 상태
		- 원인: draft의 vendor field는 역참조된 work에서 가져오는데 정말 간혹 work가 없는 draft가 생김(기안은 했으나 작업을 최종적으로 안하기로 한 것 or 변경 기안).
		- 해결: 체크박스에 체크하면 vendor 직접 입력 가능.
	- settlement 1원 차이 버그 수정
		- 문제점: 합계와 아코디언의 합계가 일치하지 않음
		- 원인: field value가 -1인 work가 존재(아마도 `input type="number"`에서 마우스 휠로 오작동 된 것으로 추정). 기존에는 아코디언에 `__gt:0` filter를 사용해서 0이상인 것만 보이게 했으나 DB 집계에는 음수도 포함되어 발견됨. 
		- 해결: 아코디언 필터를 `!=0`으로 변경해 음수도 포함.
